## Delta Analysis

### `SC2124` is currently being hit by array/string assignment cases `[shuck-fix]`

**Bucket:** ShellCheck-only  
**Impact:** 11 locations / 11 scripts in the sample

The sampled `SC2124` diagnostics are not about piping into `kill`. They are all ShellCheck array-to-string / word-splitting warnings in ordinary assignments and command arguments, while `pipe_to_kill.rs` currently only looks for pipelines that end in `kill`. That means C046 is not just a little broad or narrow: the rule implementation is aimed at a different shape than the one the corpus is actually exercising for SC2124.
