### Redirected Command Substitutions `[shuck-fix]`

**Bucket:** Shuck-only  
**Impact:** 18 locations / 14 scripts

Shuck reports `SC2256` when a command substitution contains redirects that discard or reroute output inside the subshell, but ShellCheck stayed silent on every sampled location. The same shape shows up in small direct examples, generated build scripts, and project-closure-heavy helper wrappers, which suggests the current matcher is broader than ShellCheck's sampled behavior rather than just pointing at the wrong span.
